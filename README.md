# Holy Sheet

![alt text](https://media.giphy.com/media/W2Pjf0ZOnwFl96836I/giphy.gif)  
Features

1. Company Search
   - Name
   - Stock Ticker
2. News Carousel
3. Company Card
   - Image
   - Sector
   - Exchange
4. Company Stock and Financial Data

### Getting started

```sh
$ npm install
```

Compile Front-End Code

```sh
$ npm run build-prod
```

Start Server

```sh
$ npm run start
```

Open [localhost:80](http://localhost:80)
